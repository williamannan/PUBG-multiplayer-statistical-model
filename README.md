# PUBG-multiplayer-statistical-model

This repository contains the code supporting the paper titled **"A Bradley–Terry Framework for Estimating Team Strength in Multi-Team Battle Royale Games."**

## Contents of the Five (5) Function Files

## 1. CompHelperFunc
This function file contains all the helper functions required to run the entire simulation. These include functions for simulating match data, ranking teams based on their latent team strengths and average team sizes, fitting the logistic regression model to estimate the optimal team strengths, performing cross-validation, and other supporting tasks.

## 2. MultiCrossVal
This function file simulates multiple sets of match data (250 datasets in this study). For each simulated dataset, it performs cross-validation to determine the optimal penalty value, fits both the penalized and unpenalized models, and reports the average results, which are provided in the **Sim_Excel** folder of this repository.

## 3. Compiled_Results
This function file imports the simulation results generated by **MultiCrossVal** and produces the figures presented in the paper.

## 4. MonteCarlo_Estimate
This function performs the Monte Carlo leave-match-out estimation described in Section 4.2 of the paper.

## 5. SensitivityEstimation
This function file performs the sensitivity analysis used to determine the optimal values of the parameters $\beta$ and $\gamma$, which describe the effect of team size on the game dynamics in the elimination and pairwise models, respectively.
